# AI-Augmented To-Do List

A clean, dynamic, and interactive To-Do List web application built using **AI-generated JavaScript** and **Bootstrap 5**.

## Features

- ✅ Add tasks dynamically via input field or Enter key
- ✅ Remove tasks using a trash icon
- ✅ Placeholder message when no tasks exist
- ✅ Event delegation for efficient click handling
- ✅ Responsive and modern UI using Bootstrap
- ✅ Custom hover effects and styling via CSS

### ➕ Add a Task:
- Type a task and click **Add Task** or press **Enter**
- The task is displayed with a checkbox and delete icon

### ❌ Remove a Task:
- Click the red 🗑️ trash icon to delete the task from the list


## 📦 Technologies Used

- HTML5
- CSS3 + Bootstrap 5
- JavaScript (Vanilla)
- Bootstrap Icons
