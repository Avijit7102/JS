Goal:

- Fetch user data from: https://jsonplaceholder.typicode.com/users/1

- Display name, email, phone, and company name on the page.


What This Does:

- Makes an API request using async/await

- Waits for data to come back

- Parses it and injects it into the HTML page

- Displays a fallback message on error

- Real-time UI updates every 5 seconds
